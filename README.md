# raspi-oled-ui
# Raspberry Pi 制御システム（訓練課題ベース）

## 概要
本プロジェクトは職業訓練校の課題をベースに、
Raspberry Piを用いてGPIO入力・OLED表示・ブザー制御を組み合わせた
簡易UIシステムを再構築したものです。

状態管理（state）によって画面やボタンの動作を切り替える設計になっています。

---

## 使用技術
- Python
- RPi.GPIO（GPIO制御）
- Adafruit SSD1306（OLED制御）
- PIL（画像・文字描画）
- Raspberry Pi

---

## システム概要（動作）
本プログラムは「状態遷移（state machine）」で制御されています。

### ■ 基本動作
- whileループで常時ボタン入力を監視
- ボタン入力をフラグ化
- stateに応じて画面と動作を切り替え

---

## 状態（state）の流れ

- state = 0  
  → 待機（画面クリア）

- state = 10  
  → 初期画面表示（エアコン風UI）

- state = 20  
  → 操作モード（メイン制御）

- state = 30 / 40 / 50  
  → モード・温度変更

---

## 主な機能

### ■ 入力制御
- スイッチ4つで操作
- 割り込み処理（event_detect）使用

### ■ 表示制御
- OLEDに日本語UI表示
- 温度・モード・風量バー表示

### ■ 音フィードバック
- ブザーで操作音（PWM制御）

---

## 工夫した点

- 状態（state）によるUI制御設計
- 割り込み＋フラグによる入力処理
- OLED上での視覚的UI表現
- ピクセル単位での風量バー設計
- 実機制御を意識した構成
